
# Budget-App

App for managing your budgets


## Demo

https://blizzeq.github.io/firebase-budget/
## Run Locally

Clone the project

```bash
  git clone https://github.com/Blizzeq/firebase-budget.git
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
